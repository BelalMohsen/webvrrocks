Instructions

  1. See this post about WebVR coming to Servo.
  2. Download the experimental WebVR Servo build.
  3. Extract the RAR archive into an appropriate location.
  4. Open the Windows command line by clicking on the Windows Start menu and entering cmd.exe.
  5. Open the directory to which you extracted the archive (e.g., cd C:\%USERPROFILE%\Downloads\servo-webvr\).
  6. Run servo.exe --resources-path resources webvr\room-scale.html.
  7. When the sample page loads, click on the Enter VR button, and enjoy!
